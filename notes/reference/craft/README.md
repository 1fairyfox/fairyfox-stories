# Craft — the body of knowledge

The living reference every book is planned and written against. Fiction here is held to the
**highest craft bar** (see `CLAUDE.md`); this folder is *how* we hold it. Grounded in real
writing-community and published knowledge, not intuition — update it as understanding grows.

| Note | What it covers |
|------|----------------|
| `forms.md` | The four forms (short story · novelette · novella · novel) and each one's distinct structure — **read before planting any book** |
| `structure.md` | Story architecture: the core loop, arcs, scene/sequel, chapters, pacing |
| `character.md` | Real, flawed, agency-driven characters; anti-Mary-Sue; why the hollow reader-insert is disproven |
| `voice-and-pov.md` | First-person craft: building a true interior voice and its pitfalls |
| `what-makes-fiction-good.md` | Captivation, stakes, resonance; what well-received books share; what readers dislike |
| `pitfalls.md` | The newbie traps to never fall into |
| `serialization.md` | Writing a book a chapter at a time so it pulls across a daily-grown serial |
| `genre-range.md` | Keeping the shelf wide: genres, modes, and out-of-the-box swings |

Every book's `blueprint.md` should be traceable to these — if a plan violates a principle
here, fix the plan.
